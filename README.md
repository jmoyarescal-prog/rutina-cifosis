# Rutina Cifosis 🏠

App correctiva para cifosis postural. ~15 minutos, sin material.

## Cómo subir a Vercel (gratis)

### Paso 1 — GitHub
1. Ve a [github.com](https://github.com) y crea una cuenta si no tienes
2. Crea un nuevo repositorio llamado `rutina-cifosis` (público)
3. Sube todos estos archivos al repositorio

### Paso 2 — Vercel
1. Ve a [vercel.com](https://vercel.com) y regístrate con tu cuenta de GitHub
2. Haz clic en "Add New Project"
3. Selecciona tu repositorio `rutina-cifosis`
4. Vercel detecta automáticamente que es Vite → clic en "Deploy"
5. En ~1 minuto tienes tu URL (ejemplo: `rutina-cifosis.vercel.app`)

### Paso 3 — Instalar en el móvil
**Android (Chrome):**
- Abre la URL en Chrome
- Menú ··· → "Añadir a pantalla de inicio"

**iPhone (Safari):**
- Abre la URL en Safari
- Botón compartir → "Añadir a pantalla de inicio"

¡Ya tienes la app instalada como si fuera nativa!

## Desarrollo local
```bash
npm install
npm run dev
```
